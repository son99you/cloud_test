VestSign, SKCertService(Koscom)

VestSign은 공인인증 솔루션입니다.

지원범위

PC

  • Internet Explorer 7 이상
    • Koscom은 8이상
  • Chrome
  • FireFox
  • Safari

Mobile

  • Android 4 이상
    • 브라우저 WebKit 버전 537 이상
    • 기본브라우저 (삼성)
    • Chrome
    • FireFox
  • iOS 기본브라우저 - Safari

패키지 구성

파일 이름 설명
VestSign-${version}_#.zip 메인 패키지(난독화)

압축 파일 내부

파일 이름 설명
vestsign_euckr UTF-8 인코딩된 모듈
vestsign_utf8 UTF-8 인코딩된 모듈
vestsign_*/doc API 가이드 문서
vestsign_*/languages html별 다국어지원 언어 js(한글, 영어, 일본어, 중국어)
vestsign_*/library 모듈에 사용되는 library
vestsign_*/sample Test용 샘플 html
vestsign_*/scripts views/*.html의 js
vestsign_*/views 페이지별 html
vestsign_*/vestsign.js 고객사에서 호출할 API 목록 & config 설정
