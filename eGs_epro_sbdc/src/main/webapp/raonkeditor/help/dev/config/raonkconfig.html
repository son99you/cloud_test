K Editor 환경 설정 파일 (xml)


케이 에디터의 환경 설정 파일로 프로그램의 각 기능에 대한 정의 및 환경을 설정합니다.
프로그램 구동을 위한 필수 파일입니다.


환경 설정 파일 개요
환경 설정 파일 개요
파일명 raonkeditor.config.xml
위치 K Editor 가 설치된 루트의 config/ 폴더 아래에 위치
문서유형 XML 1.0 의 utf-8
유의사항 값 입력 시 " < " → "&lt; ", ">" → "&gt;", "&" → "&amp;" 와 같이 html entity 형태로 입력해야 합니다.
license
<license /> 제품 및 라이센스 관련 정보
product_key 제품 구입 당시 발행된 제품 고유 번호입니다.
제품 고유 번호가 유효하지 않은 경우 기능상 제한을 받을 수 있습니다.
제품 고유 번호는 XXXXX-XXXX-XXXX-XXXXX 형태로 기술되어야 합니다.
제품 테스트를 위한 평가판 제품 고유 번호는 http://www.raonk.com 에서 신청가능합니다.
license_key, license_key_html5 제품 라이센스 관리를 위한 암호화된 스트링 값입니다.
license_key는 Agent용 라이선스, license_key_html5는 html5용 라이선스 입니다.
제품 구입 시 주어집니다.
이 값이 유효하지 않은 경우 기능상 제한을 받을 수 있습니다.
에디터 환경설정
<setting /> 에디터 환경 설정
runtimes agent 설정시 agent 모드가 우선 적용됩니다.
html5 설정시 html5 모드가 우선 적용되며, html5모드는 ie10 이상만 지원합니다.
예) "agent" or "html5"
mime_use [Agent 모드 전용]
MIME 사용 여부를 설정합니다.
기본값은 "0" 이고 미사용, "1" 로 설정시 MIME 사용 입니다.
encoding 에디터 디자인 모드에 사용할 문자 인코딩(character encoding) 형태를 설정합니다.

다국어 지원을 위해서는 "utf-8" 사용을 권장하며, K Editor의 기본값도 "utf-8" 입니다.
사이트의 구성에 맞게 "euc-kr" 등을 값으로 사용할 수 있습니다.
ie_compatible IE 랜더링 모드를 설정합니다.

예) EmulateIE8 또는 8 등 버전만 입력 합니다.
doc_lang <html> 요소에 주언어를 설정합니다.

속성값은 ISO-639-1에서 지정한 언어 코드를 사용하며 한국어는 ko, 영어는 en, 일본어는 ja 등 입니다.
기본값은 "ko" 입니다.
lang 에디터 제품에 설정할 기본 언어를 설정합니다.

각 문화권(국가)의 고유한 이름을 지정해야합니다.
· ko-kr : 한국어 (대한민국)
· en-us : 영어 (미국)
· zh-cn : 중국어 (중국)
· zh-tw : 중국어 (대만)
· ja-jp : 일본어 (일본)
기본값은 "ko-kr" 입니다.
width 에디터의 넓이를 설정합니다.

넓이는 "px", "%" 모두 가능합니다.
기본값은 "900px" 입니다.
height 에디터의 높이를 설정합니다.

높이는 "px", "%" 모두 가능합니다.
기본값은 "550px" 입니다.
skinname 에디터의 스킨 색상을 설정합니다.

사용 사이트의 색상 패턴과 조합을 위해서 이 설정을 사용하면 유용할 수 있습니다.
색상으로는 blue, bluegray, bluegreen, brown, darkgray, gold, gray, green, orange, pink, purple, red, skyblue, yellow 값이 있습니다.
기본값은 "bluegray" 입니다.
font_family 에디터 로드 시 사용할 기본 폰트를 설정합니다.

K Editor는 다국어를 지원하므로 다국어 사용시 각 언어에 맞는 기본 폰트를 지정하는 기능으로 유용할 수 있습니다.
기본 폰트는 "굴림" 입니다.
font_size 에디터가 로드시 사용할 기본 폰트 크기를 설정합니다.

기본 폰트는 "12pt" 입니다.
line_height 에디터가 로드시 사용할 기본 줄간격을 설정합니다.

기본 값은 "1.2" 입니다.
show_font_real 에디터에 내용 작성 시 커서가 있는 곳의 폰트명, 크기, 줄간격을 표시할 지에 대한 설정합니다.

기본값은 "1" 이고 "0" 으로 설정시 사용 안함입니다.
accessability 에디터 디자인 모드에 사용할 웹 접근성(web accessibility)단계를 설정합니다.

장애를 가진 사람과 장애를 가지지 않은 사람 모두가 웹사이트를 이용할 수 있게 하는 방식입니다.
웹 접근성 "0"단계는 웹 접근성 미설정합니다.
"1"단계는 사용자로 하여금 표, 그림, Iframe 삽입시 웹접근성을 위반하면 경고창으로 알려준 후 다음 단계로 진행합니다.
"2"단계는 사용자로 하여금 표, 그림, Iframe 삽입시 웹접근성을 위반하면 경고창으로 알려준 후 수정될 때까지 다음단계로 진행하지 않습니다.
기본값은 "0" 입니다.
show_topmenu 에디터 메뉴바를 보이거나 숨기는 기능을 설정합니다.

기본값은 "1" 이고, "0" 로 설정시 메뉴바를 숨김으로 설정합니다.
show_toolbar 에디터의 툴바는 상단과 하단으로 구성되어 있습니다.
이 설정은 상단, 하단 툴바를 보이거나 숨기는 기능을 설정합니다.

기본값은 "3"(툴바 모두 보이기) 이고,
"0"으로 설정시 툴바를 숨김으로 설정합니다.
"1" 로 설정시 "상단" 툴바만 보이도록 설정합니다.
"2" 로 설정시 "하단" 툴바만 보이도록 설정합니다.
grouping 툴바 보기 설정시,

에디터 아이콘 툴바 그룹핑을 설정합니다.
노출하고자 하는 아이콘이 많을 경우 아이콘 그룹핑을 사용하시면 유용할 수 있습니다.
기본값은 "0" 이고, "1" 로 설정시 아이콘 툴바 그룹핑 사용합니다.
show_statusbar 에디터의 상태바는 "디자인", "HTML" 및 "미리보기", "TEXT" 탭으로 구성되어 있습니다.
이 설정은 상태바를 보이거나 숨기는 기능을 설정합니다.
기본값은 "1" 이고, "0"으로 설정시 상태바를 숨김으로 설정합니다.
loading 에디터 상태바 설정시,

"디자인", "HTML" 및 "미리보기", "TEXT" 탭 변경 시 로딩 텍스트 노출 여부를 설정합니다.
기본값은 "1" 로딩 텍스트를 노출하고, "0"으로 설정시 로딩 텍스트를 미노출 합니다.
showdialog_pos 에디터 다이얼로그창이 표시될 위치를 설정합니다.

기본값은 "1" 이고, 에디터 영역 중앙에 표시됩니다.
이외의 값이면 브라우저 중앙에 표시 됩니다.
source_viewtype 에디터 html탭에서 html소스 정렬 여부를 설정합니다.

"0"으로 설정 시 정렬안함 입니다.
"1"로 설정 시 들여쓰기 정렬 적용 입니다.
"2"로 설정 시 들여쓰기 정렬 + 줄바꿈 적용 설정입니다.
기본값은 "2"입니다.
wrap_ptag_to_source set API가 실행될 때 block 태그를 제외한 태그는 ptag로 감싸는 설정합니다.

"0"으로 설정 시 사용안함 입니다.
"1"로 설정 시 사용함 입니다.
"2"로 설정 시 사용함(내용이 없는 텍스트 노드 제외) 입니다.
기본값은 "2" 입니다.
user_css_url 설정한 css를 에디터 영역에 추가하는 설정 입니다.

url 형태의 값을 입력해야 합니다.
view_mode_auto_height View 모드시 에디터 안의 내용 만큼 자동으로 height가 늘어납니다.

기본값은 "0" 이고, "1"로 설정시 자동 height을 사용으로 설정합니다.
system_title 다이얼로그창 상단 문구 변경시 설정합니다.

기본값은 "K Editor" 이고, "0"로 설정시 사용하지 않음
그 이외의 값을 입력하시면 해당 값으로 으로 표시 됩니다.
use_ruler 눈금자 표시 사용여부를 설정합니다.

기본값은 "0" 이고 눈금자 사용 안함, "1"로 설정시 눈금자 사용으로 설정합니다.
ruler_init_position 눈금자 사용 설정시,

눈금자 포인터의 위치 값(픽셀)을 설정합니다.
기본값은 "680px" 입니다.
view_pointer 눈금자 사용 설정시,

눈금자 포인터 아이콘 표시 사용여부를 설정합니다.
기본값은 "1" 이고 표시, "0"로 설정시 포인트 아이콘을 표시 안함으로 설정합니다.
view_guide_line 눈금자 사용 설정시,

에디터 작성 영역에 눈금자 포인터 기준선(세로) 표시 사용여부를 설정합니다.
기본값은 "1" 이고 표시, "0"로 설정시 눈금자 포인트 기준선 표시 안함으로 설정합니다.
view_ruler 눈금자 사용 설정시,

에디터 작성 영역 상단에 눈금자 이미지 표시 사용여부를 설정합니다.
기본값은 "1" 이고 표시, "0"로 설정시 눈금자 이미지 표시 안함으로 설정합니다.
auto_body_fit 별도 지정한 에디터 작성영역 크기에 맞게 입력한 텍스트가 자동으로 줄바꿈 되도록 하는 설정 값입니다.
가로 눈금자 설정과 같이 사용하는 경우 설정한 눈금자만큼 크기가 결정됩니다.
"0"로 설정시 사용 안함이고, "1"로 설정시 줄내림 사용으로 설정합니다.
기본값은 "0"입니다.
use_horizontal_line 가로 구분선 표시 사용여부를 설정합니다.

"0"로 설정시 사용 안함, "1"로 설정시 가로 구분선 표시 사용으로 설정합니다.
기본값은 "0" 입니다.
url 가로 구분선 사용 설정시 기준선 이미지 경로를 설정합니다.
http://를 포함한 전체 이미지 경로를 입력합니다.
image_base_url 에디터 이미지 기본 주소를 설정합니다.

url은 http://를 포함한 전체 경로를 입력해야 합니다.
예) http://localhost/raonkeditor/
context_menu_disable 에디터 안에서 마우스 우클릭 시 메뉴 표시 사용여부를 설정합니다.

기본값은"0" 표시, "1"로 설정시 우클릭 메뉴 표시 안함으로 설정합니다.
enter_tag Enter 키 눌렀을 때 사용하게 될 태그를 설정합니다.

기본값은 "p" 태그이며, "br" 태그로 설정 할 수 있습니다.
table_property 테이블 생성에 대한 기본값을 설정합니다.
allow_inoutdent_text 표 안에서 들여쓰기/내어쓰기 적용 시,
값이 "0"이면 표 전체 들여쓰기, 내어쓰기로 적용
값이 "1"이면 표 안에 있는 텍스트를 들여쓰기, 내어쓰기 적용 입니다.
기본값은 "0" 입니다.
width, height 생성될 테이블의 너비, 높이를 설정합니다.
class 생성 될 때 기본으로 적용될 class를 설정하는 항목입니다.
inoutdent 표 들여쓰기, 내어쓰기 시 적용될 px 값을 설정합니다.
기본값은 "20px" 이고, 기타 원하는 값으로 설정 가능합니다.
단위는 px단위로 입력해야 합니다.
init_inoutdent 표 생성 시 들여쓰기 할 기본 px값을 설정합니다.
기본값은 "0px"" 이고, 기타 원하는 값으로 설정 가능합니다.
단위는 px단위로 입력해야 합니다.
td_height 표 생성시 td 기본 높이 값을 설정합니다.
기본값은 "20px"이고 기타 원하는 값으로 설정 가능합니다.
단위는 px단위로 입력해야 합니다.
row_max_count 표 삽입시 행의 최대 값을 설정합니다.
기본 최대값은 "13" 입니다.
col_max_count 표 삽입시 열의 최대 값을 설정합니다.
기본 최대값은 "13" 입니다.
> class_list 이 항목에 class 명을 입력하면 사용자는 이 항목에 명시되어 있는 클래스 중 하나를 선택하여 적용할 수 있습니다.
구분자는 콤마(,)로 여러 개의 값을 넣을 수 있습니다.
> list_style 테이블에 선모양 기본 스타일을 설정합니다.

기본값은 solid, dotted, dashed, double, groove, ridge, inset, outset, none 이고
원하지 않는 값은 제거할 수 있습니다.
> no_resize_class 테이블 크기 조정을 하지 못하게 하는 class명 입니다.
이 항목에 지정된 class가 있는 테이블이 있다면 테이블 크기 조정이 되지 않습니다.
> show_line_for_border_none 테이블에 선이 없을 경우(선두께:0) 가상의 선을 보여줄지에 대한 설정합니다.

기본값은 "0" 이고 선이 보이지 않음으로 설정됩니다.
"1"로 설정시 점선으로 설정됩니다.
"2"로 설정시 점선으로 설정되고, 붙여넣기 할 때 에도 점선으로 설정됩니다.
set_default_style 에디터에 작성된 내용을 get 할 때에 에디터의 기본 스타일을 각 태그에 인라인으로 적용하여 리턴해 주는 설정입니다.
기본값은 "사용함"이고 "0"으로 설정 시 사용안함으로 설정됩니다.
set_style 에디터에 html소스가 set될 때에 에디터의 기본 스타일 css을 추가하는 설정입니다.
기본값은 "사용함"이고 "0"으로 설정 시 사용안함으로 설정됩니다.
body_id 에디터 body 영역에 id 값 설정합니다.

기본값은 "keditor_body"이고, "사용자 설정값"으로 설정합니다.
paste_to_image 이미지 붙여넣기가 되는 경우 별도의 이미지 붙여넣기 팝업창을 띄울지에 대한 설정입니다.
기본값은 "1" 사용함이고 "0"으로 설정 시 사용하지 않습니다.
view_mode_browser_menu 에디터 View 모드 시 우클릭 했을 때 브라우저의 우측 메뉴를 열리게 합니다.
기본값은 "1" 사용함이고, "0"로 설정시 우클릭 시 메뉴가 뜨지 않습니다.
view_mode_allow_copy 키보드 단축키 ctrl+c 사용을 허용 설정합니다.

기본값은 "1"이고, "0"로 설정시 ctrl+c 미허용으로 설정합니다.
drag_and_drop_allow 드래그 앤 드랍으로 이미지를 추가할 수 있습니다.
기본값은 "1" 사용함이고, "0"으로 사용 시 드래그 앤 드랍 이미지 추가 기능이 비활성화 됩니다.
formlist_url 에디터 템플릿 양식 파일경로를 설정합니다.

http://를 포함한 전체 경로를 설정합니다.
default_mode 에디터를 edit, view 모드로 설정합니다.
