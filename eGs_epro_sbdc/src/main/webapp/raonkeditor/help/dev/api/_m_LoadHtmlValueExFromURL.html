K Editor :: LoadHtmlValueExFromURL

void LoadHtmlValueExFromURL(htmlURL, editor_id)

에디터 디자인 영역에 지정한 경로의 html 페이지를 띄웁니다.

return value

없음.

parameters

htmlURL  에디터에 띄울 html페이지의 url을 의미합니다.
editor_id  html 페이지를 띄울 에디터의 id를 의미합니다.

remarks

에디터를 1개만 생성했을 경우 editor_id를 입력하지 않아도 이미지 태그를 가져올수 있습니다.
예) id가 editor1인 에디터를 생성했을 경우
     RAONKEDITOR.LoadHtmlValueExFromURL(htmlURL, "editor1"), RAONKEDITOR.LoadHtmlValueExFromURL(htmlURL)  
     모두 editor1 에디터 디자인 영역에 html페이지를 보여줍니다.

에디터를 2개 이상 생성했을 경우 editor_id를 반드시 입력해야 합니다.
예) id가 editor1, editor2인 에디터를 생성했을 경우
     RAONKEDITOR.LoadHtmlValueExFromURL(htmlURL, "editor1")  editor1 에디터 디자인 영역에 html페이지를 보여줍니다.
     RAONKEDITOR.LoadHtmlValueExFromURL(htmlURL, "editor2")  editor2 에디터 디자인 영역에 html페이지를 보여줍니다.
