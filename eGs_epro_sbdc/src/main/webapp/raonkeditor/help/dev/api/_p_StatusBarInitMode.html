K Editor :: Config :: StatusBarInitMode

StatusBarInitMode

에디터 "디자인", "HTML", "미리보기", "TEXT" 중 에디터 로드시 보여질 모드를 설정합니다.

remarks

기본값은 "design" 이고, HTML은 "source", 미리보기는 "preview" ,TEXT는 "text" 으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
                 
        var paramObj = {
            Id: 'editor1',          
            // 에디터 로드시 HTML 모드로 설정합니다.
            StatusBarInitMode: 'source'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
