K Editor :: Config :: EditorTabDisable

EditorTabDisable

디자인영역에서 tab키 입력시 공백 입력을 설정합니다.

remarks

기본값은 "0" 이고, "1" 로 설정시 tab키를 입력하면 다음 컨트롤로 포커스를 이동합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
                 
        var paramObj = {
            Id: 'editor1',            
            // 에디터 tab키 입력하면 다음 컨트롤로 포터스 이동을 설정합니다.
            EditorTabDisable: '1'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
