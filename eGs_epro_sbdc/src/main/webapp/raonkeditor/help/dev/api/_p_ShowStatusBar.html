K Editor :: Config :: ShowStatusBar

ShowStatusBar

에디터의 상태바는 "디자인", "HTML" 및 "미리보기", "TEXT" 탭으로 구성되어 있습니다.
이 설정은 상태바를 보이거나 숨기는 기능을 설정합니다.
사용 용도에 따라서 상태바를 숨김으로 원할 때 사용하면 유용 할 수 있습니다.

remarks

기본값은 "1" 이고, "0" 로 설정시 상태바를 숨김으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 상태바를 숨김으로 설정합니다.
            ShowStatusBar: '0'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
