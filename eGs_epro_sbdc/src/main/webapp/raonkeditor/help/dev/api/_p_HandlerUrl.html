K Editor :: Config :: HandlerUrl

HandlerUrl

웹에서 RFC 1867 표준에 따르는 POST 방식의 파일 전송을 받아주는 페이지 URL입니다.
이미지 파일 또는 Flash, media 파일 등 서버에 파일이 업로드 되지 않을 경우 이 값의 설정을 확인하세요.

remarks

기본값은 "" 입니다.
raonkeditor/handler/ 아래의 각 언어에 맞는 페이지를 호출합니다.
상대 및 절대 경로 모두를 지원합니다.

지정된 URL의 동작을 확인 하려면 handler_url URL에
"?raonk=test"를 붙여서 테스트 후 페이지에 "Hi, RAON K Editor !!!! test" 가 표시되면 정상적으로 설정된 경우입니다.

각 개발 언어에 맞는 값으로 변경 설정하세요.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 handler url을 설정합니다.
            HandlerUrl: 'http://www.raonk.com/handler/upload_handler.jsp'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
