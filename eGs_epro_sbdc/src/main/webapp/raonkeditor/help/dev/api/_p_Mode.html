K Editor :: Config :: Mode

Mode

에디터를 edit, view 모드로 설정합니다.

remarks

기본값은 "edit" 모드로 설정됩니다.

setEditorMode 메소드로 view, edit모드로 변환 할 수 있습니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">
            
        var paramObj = {
            Id: 'editor1',          
            // 에디터를 view 모드로 설정합니다.
            Mode: 'view'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
