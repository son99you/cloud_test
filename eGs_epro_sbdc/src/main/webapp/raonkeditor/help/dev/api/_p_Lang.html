K Editor :: Config :: Lang

Lang

에디터에 로드 할 언어를 설정합니다.

remarks

기본값은 "ko-kr : 한국어" 로 설정됩니다.
필요시 Lang: '언어코드'로 설정합니다.

언어코드는
ko-kr : 한국어(대한민국)
en-us : 영어(미국)
zh-cn : 중국어(중국)
zh-tw : 중국어(대만)
ja-jp : 일본어

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 언어를 en-us로 설정합니다.
            Lang: 'en-us'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
