K Editor :: Config :: SkinName

SkinName

에디터의 스킨 색상을 설정합니다.

remarks

기본값은 "bluegary" 색상으로 설정 됩니다.
필요시 SkinName: '색상명' 으로 설정합니다.

스킨 색상은
blue, bluegray, bluegreen, brown, darkgray, gold, gray, green, orange, pink, purple, red, skyblue, yellow
14가지 입니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">
            
        var paramObj = {
            Id: 'editor1',          
            // 에디터의 스킨을 gray 로 설정합니다.
            SkinName: 'bluegray'
        };
        
        new RAONKEditor(paramObj);
    
    </script>           
</div>
