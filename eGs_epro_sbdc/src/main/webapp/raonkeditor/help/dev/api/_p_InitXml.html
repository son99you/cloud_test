K Editor :: Config :: InitXml

InitXml

에디터에 로드 할 xml파일을 설정합니다.

remarks

기본값은 "raonkeditor.config.xml" 으로 설정됩니다.
필요시 InitXml: 'xml 파일명' 으로 설정합니다.

파일 경로는 raonkeditor/config 입니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터에 example.xml 으로 로드합니다.
            InitXml: 'example.xml'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
