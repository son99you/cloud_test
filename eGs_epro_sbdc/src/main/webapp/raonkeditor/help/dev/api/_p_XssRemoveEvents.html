XssRemoveEvents

에디터에 입력되면 삭제할 이벤트를 설정합니다.

remarks

onclick,onload,onchange 등 형태로 이벤트를 설정합니다.
all 로 입력하면 모든 이벤트 값을 설정합니다.
사용자가 태그에 해당되는 이벤트를 입력한 경우 에디터에서 삭제합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터에 onclick 가 입력되면 삭제합니다.
            XssRemoveEvents: 'onclick'
        };
        
        new RAONKEditor(paramObj);
    
    </script>       
</div>
