K Editor :: Config :: LineHeightList

LineHeightList

에디터 줄간격 메뉴에 노출할 줄간격 리스트를 설정합니다.

remarks

기본 줄간격은 "1.2" 입니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 메뉴에 노출할 줄간격을 1.2, 1.3, 1.4 으로 설정합니다.
            LineHeightList: '1.2,1.3,1.4'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
