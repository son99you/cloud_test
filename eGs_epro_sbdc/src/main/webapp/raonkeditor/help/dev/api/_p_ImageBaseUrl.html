K Editor :: Config :: ImageBaseUrl

ImageBaseUrl

에디터 이미지 기본 주소를 설정합니다.

remarks

url은 http://를 포함한 전체 경로를 입력해야 합니다.
예) http://localhost/raonkeditor/

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
            
        var paramObj = {
            Id: 'editor1',          
            // 에디터 이미지 기본 주소를 설정합니다.
            ImageBaseUrl: 'http://www.raonk.com/raonkeditor/'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
