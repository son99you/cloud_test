K Editor :: Config :: EnterTag

EnterTag

Enter 키 눌렀을 때 사용하게 될 태그를 설정합니다.

remarks

기본값은 "p" 태그이며, "br" 태그 등을 지정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',            
            // 에디터 Enter 키 눌렀을 때 br 태그로 설정합니다.
            EnterTag: 'br'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
