SaveFolderNameRule

이미지 및 파일을 받아서 처리하는 핸들러에 설정된 폴더 하위 경로 저장 체계를 설정합니다.

remarks

기본값은 "YYYY/MM/" 입니다.

설정 방법은
사용하지 않음 => "/",
년도 => "YYYY/",
년도/월 => "YYYY/MM/",
년도/월/일 => "YYYY/MM/DD/"
기타값 -> 해당 값으로 폴더가 생성됩니다.
폴더가 존재하지 않으면 자동으로 생성되며, 서버의 날짜가 기준이 됩니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 폴더들에 대한 저장 체계를 년/월/일 로 설정합니다.
            SaveFolderNameRule: 'YYYY/MM/DD/'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
