K Editor :: Config :: EditorHolder

EditorHolder

에디터를 특정 영역에서 생성할 수 있도록 합니다.

remarks

에디터를 생성할 영역의 id값을 입력합니다.
필요시 EditorHolder: '아이디값' 으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>
	
<script type="text/javascript">

    function fn_load() { 
            
        var paramObj = {
            Id: 'editor1',            
            // 에디터에 생성할 객체의 ID를 editor_area 로 설정합니다.
            EditorHolder: 'editor_area'
        };
        
        new RAONKEditor(paramObj);
    }

</script>

<!-- ..... 생략 ..... -->
 
<body onload="fn_load()">     
    
    <!-- ..... 생략 ..... -->       
    
    <div id="editor_area">

    <!-- 에디터를 생성할 영역입니다. -->

    </div> 
</body>
