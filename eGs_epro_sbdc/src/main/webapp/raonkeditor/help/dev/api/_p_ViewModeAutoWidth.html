K Editor :: Config :: ViewModeAutoWidth

ViewModeAutoWidth

View 모드시 에디터 안의 내용 만큼 자동으로 width가 늘어납니다.

remarks

기본값은 "0" 이고, "1"로 설정시 자동 width 사용으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 자동 width을 사용으로 설정합니다.
            ViewModeAutoWidth: '1'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
