K Editor :: Config :: InitVisible

InitVisible

에디터를 화면에 표시할지 여부를 설정합니다.

remarks

기본값은 "true" 으로 설정됩니다.
필요시 InitVisible: (true 또는 flase)으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // editor1 에디터는 화면에 표시됩니다.
            InitVisible: false
        };
        
        new RAONKEditor(paramObj);

    </script>  

    <script type="text/javascript">
            
            var paramObj = {
                Id: 'editor2',          
                // editor2 에디터는 화면에 표시되지 않습니다.
                InitVisible: true
            };
        
            new RAONKEditor(paramObj);

    </script>  
     
</div>
