K Editor :: Config :: RemoveComment

config.RemoveComment

에디터에 내용이 들어올 때 주석 삭제 여부를 설정합니다.

remarks

기본값은 "1" 주석 삭제, "0" 으로 설정시 주석을 미삭제 합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
                 
        var paramObj = {
            Id: 'editor1',          
            // 에디터에 들어온 내용 중 주석을 미삭제합니다. -->
            RemoveComment: '0'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
