K Editor :: Config :: Width

Width

에디터의 너비값을 설정합니다.

remarks

기본값은 "900px" 으로 설정됩니다.
필요시 Width: '너비값' 으로 설정합니다.

sample code

  
<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
     
<!-- ..... 생략 ..... -->
     
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터의 너비값을 750px 으로 설정합니다.
            Width: '750px'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
