K Editor :: Config :: SetValueObjId

SetValueObjId

textarea 에 id값을 설정하면 에디터 로드 후 textarea 내용이 에디터에 세팅됩니다.

remarks

필요시 SetValueObjId: '아이디값' 으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 로드 후 textarea 내용이 에디터에 세팅 될 ID값을 설정합니다.
            SetValueObjId: 'raonk'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
