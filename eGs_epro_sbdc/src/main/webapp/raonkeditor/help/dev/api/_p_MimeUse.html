K Editor :: Config :: MimeUse

MimeUse

MIME 사용 여부를 설정합니다.

remarks

기본값은 "0" 이고 미사용, "1" 로 설정시 MIME 사용 입니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',
            // 플러그인 모듈에서 MIME 을 사용으로 설정합니다.
            MimeUse: '1'
        };

        new RAONKEditor(paramObj);

    </script>       
</div>
