K Editor :: Config :: ShowDialogPosition

ShowDialogPosition

에디터 다이얼로그창이 표시될 위치를 설정합니다.

remarks

기본값은 "1" 이고, 에디터 영역 중앙에 표시됩니다.
이외의 값이면 브라우저 중앙에 표시 됩니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 다이얼로그창 위치를 브라우저 중앙에 설정합니다.
            ShowDialogPosition: '0'
        };
        
        new RAONKEditor(paramObj);

    </script>
</div>
