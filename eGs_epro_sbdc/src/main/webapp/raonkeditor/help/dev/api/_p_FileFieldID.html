K Editor :: Config :: FileFieldID

FileFieldID

파일 업로드에 사용되는 File 태그의 name을 설정합니다.

remarks

기본값은 'Filedata' 로 설정됩니다.
필요시 FileFieldID: 'newName' 으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',            
            // 파일 업로드 File 태그의 name을 raonk로 변경합니다.
            FileFieldID: 'raonk'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
