K Editor :: Config :: Runtimes

Runtimes

에디터 모드를 설정합니다.
(에이전트모드, 웹표준모드 중 선택)

remarks

agent 설정시 agent 모드가 적용됩니다.
html5 설정시 html5 모드가 적용되며, html5모드는 ie10 이상만 지원합니다
(html5가 지원되지 않는 브라우저로 접속을 하게 되면 에디터에서 자동으로 agent모드로 구동됩니다.) 예) "agent" or "html5"

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터를 agent 모드로 설정합니다.
            Runtimes: 'agent'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
