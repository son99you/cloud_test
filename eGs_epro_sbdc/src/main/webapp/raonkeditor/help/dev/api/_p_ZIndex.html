K Editor :: Config :: ZIndex

ZIndex

에디터 배치 순서를 설정합니다.
숫자가 높을 수록 앞에 설정됩니다.

remarks

기본값은 "10000" 입니다.
필요시 zIndex: '배치 순서값' 으로 설정합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',          
            // 에디터 배치 순서를 11000 으로 설정합니다.
            ZIndex: '11000'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
