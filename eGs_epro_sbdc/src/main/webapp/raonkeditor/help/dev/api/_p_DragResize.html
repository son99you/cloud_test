DragResize

에디터 또는 브라우저의 기본 리사이즈 사용여부를 설정합니다.

remarks

기본값은 "0" 브라우저 기본 리사이즈를 사용하고, "1"로 설정시 에디터에서 제공하는 리사이즈를 사용합니다.

"1" 일 경우 DragResizeApplyBrowser 를 설정 가능합니다.
DragResizeApplyBrowser에 입력한 브라우저만(ie,firefox,opera,chrome,safari) 리사이즈를 사용합니다.
빈값일경우 모든 브라우저가 설정됩니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',
            // 에디터에서 제공하는 리사이즈를 사용 설정합니다.
            DragResize: '1',
            // 에디터에서 제공하는 리사이즈 브라우저는 ie,firefox,opera 만 설정합니다.
            DragResizeApplyBrowser: 'ie,firefox,opera'
        };
        
        new RAONKEditor(paramObj);

    </script>       
</div>
