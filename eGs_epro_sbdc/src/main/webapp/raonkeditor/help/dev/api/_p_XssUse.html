XssUse

에디터 XSS 방지를 설정합니다.
1단계는 script, vbscript 태그 입력시 문자열로 처리하고, src, href 등에 입력된 보안위협요소를 제거합니다.

remarks

기본값은 "1" 이고, "0" 로 설정시 XSS 방지 미설정 합니다.

sample code

<script type="text/javascript" src="keditor/js/raonkeditor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        var paramObj = {
            Id: 'editor1',
            // 에디터 XSS 방지 미설정 합니다.
            XssUse: '0'
        };

        new RAONKEditor(paramObj);

    </script>       
</div>
